Сортировка пузырьком

  • быстродействие в худшем случае: О(n2) сравнений, О(n2) обменов
  • быстродействие в лучшем случае: О(n) сравнений, О(1) обменов
  • среднее быстродействие: О(n2) сравнений, О(n2) обменов

Реализация на Python

def bubble_sort(array):
    for i in range(len(array)):
        for j in range(len(array) - i - 1):
            if array[j + 1] < array[j]:
                temp = array[j]
                array[j] = array[j + 1]
                array[j + 1] = temp
    return array
